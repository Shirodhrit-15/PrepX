## Step 1 — Update pubspec.yaml

Replace your pubspec.yaml with this (all packages uncommented):

```yaml
dependencies:
  flutter:
    sdk: flutter
  cupertino_icons: ^1.0.8
  firebase_core: ^2.32.0
  firebase_auth: ^4.20.0
  cloud_firestore: ^4.17.5
  firebase_storage: ^11.7.7
  cloud_functions: ^4.7.6
  provider: ^6.1.2
  file_picker: ^8.1.2
  image_picker: ^1.1.2
  permission_handler: ^11.3.1
  http: ^1.2.2
  web_socket_channel: ^3.0.1
  google_sign_in: ^6.2.1
  uuid: ^4.4.0
  intl: ^0.19.0
  shared_preferences: ^2.3.2
```

Then run:
```bash
flutter pub get
```

---

## Step 2 — Create Firebase Project

1. Go to https://console.firebase.google.com
2. Click **Add Project** → name it `prepx`
3. Enable **Google Analytics** (optional)
4. In Project Settings → add apps:
   - **Android**: use your package name (e.g. `com.example.prepx`)
   - **iOS**: use your bundle ID

---

## Step 3 — Enable Firebase Services

In the Firebase Console:

| Service | Where | Settings |
|---|---|---|
| **Authentication** | Build → Authentication | Enable Email/Password + Google |
| **Firestore** | Build → Firestore | Create in **production** mode |
| **Storage** | Build → Storage | Create with default rules |
| **Functions** | Build → Functions | Requires Blaze (pay-as-you-go) plan |

---

## Step 4 — Connect Flutter to Firebase

```bash
# Install FlutterFire CLI
dart pub global activate flutterfire_cli

# Configure (auto-generates firebase_options.dart)
flutterfire configure --project=YOUR_FIREBASE_PROJECT_ID
```

This creates `lib/firebase_options.dart` automatically. **Delete the template version** provided in this repo.

---

## Step 5 — Android Setup

In `android/app/build.gradle`:
```gradle
android {
    compileSdkVersion 34
    defaultConfig {
        minSdkVersion 21   // ← Firebase requires 21+
    }
}
```

In `android/app/src/main/AndroidManifest.xml`, add microphone permission:
```xml
<uses-permission android:name="android.permission.RECORD_AUDIO"/>
<uses-permission android:name="android.permission.INTERNET"/>
```

---

## Step 6 — iOS Setup

In `ios/Runner/Info.plist`, add:
```xml
<key>NSMicrophoneUsageDescription</key>
<string>PrepX needs microphone access for AI voice interviews</string>
<key>NSPhotoLibraryUsageDescription</key>
<string>PrepX needs photo access for profile picture</string>
```

---

## Step 7 — Deploy Cloud Functions

```bash
# Navigate to functions folder
cd functions

# Install dependencies
npm install

# Set API keys as environment variables
firebase functions:secrets:set GEMINI_API_KEY
# (paste your key when prompted)

firebase functions:secrets:set VAPI_API_KEY
# (paste your Vapi private key when prompted)

# Deploy
firebase deploy --only functions
```

---

## Step 8 — Deploy Firestore Rules + Indexes

```bash
firebase deploy --only firestore
firebase deploy --only storage
```

---

## Step 9 — Add Your Keys

### Vapi Setup
1. Sign up at https://vapi.ai
2. Get your **Public Key** (for Flutter WebSocket) and **Private Key** (for Cloud Functions)
3. In `lib/services/vapi_service.dart`, replace:
   ```dart
   static const String _vapiPublicKey = 'YOUR_VAPI_PUBLIC_KEY';
   ```

### Gemini Setup
1. Get API key from https://aistudio.google.com/app/apikey
2. Set it via Firebase secrets:
   ```bash
   firebase functions:secrets:set GEMINI_API_KEY
   ```

flutter run

## Firestore Collections Reference

### `users/{uid}`
```
displayName: string
email: string
photoUrl: string?
targetRole: string?
experienceLevel: 'fresher' | 'junior' | 'mid' | 'senior'
resumeUrl: string?
totalSessions: number
avgScore: number
createdAt: timestamp
```

### `sessions/{sessionId}`
```
userId: string
jobRole: string
domain: 'technical' | 'hr' | 'behavioral' | 'mixed'
difficulty: 'easy' | 'medium' | 'hard'
vapiCallId: string?
status: 'pending' | 'active' | 'completed' | 'failed'
durationSeconds: number
createdAt: timestamp
endedAt: timestamp?
```

### `results/{sessionId}`
```
userId: string
overallScore: number (0-100)
transcript: string
questionResults: array of { question, answer, score, feedback, idealAnswer }
strengths: array of strings
improvements: array of strings
generalFeedback: array of strings
categoryScores: { communication, technical, problemSolving, confidence }
createdAt: timestamp
```

### `questions/{questionId}`
```
domain: string
level: string
text: string
tags: array of strings
hint: string?
```
